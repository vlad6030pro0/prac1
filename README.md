# Калькулятор
## Ограничения
 - 6-9 знаков после запятой при взаимодействии с дробными числами
 - Валидация данных
 - Деление на ноль

## Ошибки
 - При вводе "exit" в начале работы программы, она не завершится, а будет ожидать ввода числа
 - Отсутствует проверка на положительность вводимого числа при операции с корнем
 - Команды чувствительны к регистру
